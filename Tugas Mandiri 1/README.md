# Sistem Survei Kepuasan Pasien

Website ini merupakan tugas praktikum Pemrograman Web yang dibuat menggunakan HTML, CSS, dan JavaScript.

## Fitur Website
- Form data pasien
- 10 pertanyaan survei kepuasan
- Kolom kritik dan saran
- Pesan sukses setelah survei dikirim

## Teknologi yang digunakan
- HTML
- CSS
- JavaScript

## Struktur File

index.html
Halaman utama website yang berisi form survei.

style.css
Digunakan untuk mengatur tampilan website agar lebih menarik.

script.js
Digunakan untuk memberikan interaksi pada website seperti menampilkan pesan ketika survei dikirim.

## Cara Menjalankan Website

1. Download repository ini
2. Buka file index.html di browser
3. Isi form survei
4. Klik tombol Kirim Survei
